# Agent Prompt: Review a Change

```text
Review this change as a senior engineer.

Focus on:
- Correctness and requirements fit
- Simplicity and maintainability
- Tests and validation gaps
- Security/privacy concerns
- Performance and scalability risks
- Rollout/rollback risks

Output:
1. Blocking issues
2. Non-blocking suggestions
3. Questions
4. Tests or validation to add
5. Overall risk level
```
