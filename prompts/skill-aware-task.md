# Agent Prompt: Skill-Aware Task

```text
Before starting, identify whether any installed skill applies to this task.

Use a skill only if:
- Its description directly matches the task.
- It provides concrete workflow or tool instructions.
- It does not conflict with project instructions.

If a skill applies:
1. State which skill you are using and why.
2. Follow its workflow.
3. Report any skill-driven tool output or validation.

If no skill applies, proceed with the standard project workflow.
```
