# Workflow Maintenance

Review this repo after major launches, incidents, or repeated review comments.

Ask:

- Which checklist would have caught this earlier?
- Which prompt needs more context or guardrails?
- Which validation command should become standard?
- Which decision should be captured as an ADR?
- Which manual step should be automated?

Keep workflow changes small and dated through PRs.
